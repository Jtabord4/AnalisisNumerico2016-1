Help For System of linear Equations

Direct Methods

Steps:
1. Fill the values field with the number of sistems of equations
2. Press get matrix
3. Use the equations given and fill the matrix
4. Select the appropiate method
5. Run that bitch!

Iterative Methods

Steps:
1. Fill the values field with the number of sistems of equations
2. Press get matrix
3. Use the equations given and fill the matrix
4. Provide the number of iterations and TOL
5. Select the appropiate method
6. Run that bitch!
